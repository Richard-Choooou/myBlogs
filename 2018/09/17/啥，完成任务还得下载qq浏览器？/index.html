啥，完成LOL任务还得下载qq浏览器？

发表于 2018-09-17

前言

腾讯最近开启了七周年活动，要完成一系列任务，作为LOL铁粉自然要肝到底，今天好不容易开启了最后一个任务，结果内容让我大跌眼镜，居然是推广一系列其他平台，其中有一项是，用QQ浏览器打开活动页面，作为一个有追求的前端er,自然是拒绝国产浏览器的，于是寻找使用技术手段解决问题的办法，接着一篇水文就这样诞生啦。

对于技术层面的思考

特定浏览器打开特定页面，服务器貌似除了user agent就没办法识别用户使用的浏览器了，那么就从user-agent下手，于是百度查找相关修改ua的办法，看了几页也没看见合适的，大部分要安装插件，于是上google碰碰运气，结果还真给我找到了。

修改方法

  • 进入特定的活动页面

http://event.browser.qq.com/stdl/activity/lol_warnight/index.html?ADTAG=lolgw

  • 打开chrome控制台，在more tools中找到 network conditions

  • 取消select automatically 的勾选
  • 在下面的输入框中 填写qq浏览器的user agent

Mozilla/5.0 (Windows NT 6.1; WOW64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/47.0.2526.80 Safari/537.36 Core/1.47.277.400 QQBrowser/9.4.7658.400

  • 在不关闭控制台的情况下刷新页面

  • 最后一步也是最重要的一步点击领取按钮

任务完成！！！

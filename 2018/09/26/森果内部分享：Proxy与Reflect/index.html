proxy

proxy顾名思义–代理，可以给一个对象添加代理，从而实现你想要追加的功能，vue3.0将会使用proxy函数作为对象的相应拦截器，到时候可以很方便的使用set以及map集合。 proxy接收两个参数，第一个是需要代理的对象，第二个参数是代理的配置项。如下所示：

代理配置项

get

用于拦截对象属性的读取

set

用于拦截对象属性的设置，可以用于设置对象的属性为只读。

has

用于判断对象是否有某个属性

apply

用于拦截函数的执行，可以很方便的给函数添加其他功能

打包 nodejs c++ 扩展环境安装

首先下载 visual studio 注意不是 vscode

接着安装 node-gyp

创建空文件夹，在根目录新建 binging.gyp 文件，内容如下：

target_name 表示打包后的文件名，sources 标明需要包含的 c++ 代码。

接着运行命令行代码

node-gyp configure

此时可能会报错

大部分的原因在于没有 安装 python2，此时需要安装 python2

安装完后再次运行 node-gyp configure，此时可能还会报错，如下：

此时需要安装 Windows-Build-Tools

npm install –global windows-build-tools

此时打开 visual studio installer，可以发现在自动安装 15 版本的生成工具，等它安装完。

安装完成后，继续运行 node-gyp configure
